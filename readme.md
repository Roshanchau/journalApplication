    # Spring Boot Learning Notes

This repository contains learning resources and notes for Java backend development, focusing on Spring Boot, the Spring Framework, and Maven.

## 📚 Learning Resources

### 1. Introduction to Spring Boot

Learn what Spring Boot is, its features, advantages, architecture, and why it is widely used for building modern Java applications.

* https://www.geeksforgeeks.org/springboot/introduction-to-spring-boot/

### 2. Difference Between Spring and Spring Boot

Understand the differences between the Spring Framework and Spring Boot, including configuration, setup, and development advantages.

* https://www.geeksforgeeks.org/java/difference-between-spring-and-spring-boot/

### 3. What is Maven?

Learn about Maven, its project structure, dependency management, build lifecycle, and its role in Java application development.

* https://www.geeksforgeeks.org/advance-java/what-is-maven/

---

## 🎯 Learning Objectives

* Understand the fundamentals of Spring Boot.
* Learn the differences between Spring and Spring Boot.
* Understand Maven and dependency management.
* Become familiar with Java backend project structure and best practices.
